# AI-Powered UGC Ad Content Pack

## Project Overview

This project demonstrates the use of Artificial Intelligence to create a complete User Generated Content (UGC) advertising campaign for a skincare product.

The selected product is **Himalaya Purifying Neem Face Wash**. Using AI-powered prompt engineering, multiple marketing assets were generated including hooks, ad scripts, CTAs, captions, and promotional visuals.

---

## Product Chosen

**Himalaya Purifying Neem Face Wash**

A herbal face wash enriched with Neem and Turmeric that helps remove impurities, control excess oil, and support acne-prone skin.

---

## Objectives

- Generate engaging marketing hooks
- Create UGC-style advertisements
- Write conversion-focused CTAs
- Develop platform-specific ad content
- Create promotional visuals
- Learn prompt engineering techniques
